# Basic-Somewhat-HTTP-Server

BSHS is a simple HTTP Server designed for ease of use. It has simple command line options, and you can run it from anywhere.

There are 2 dependencies that you can find on my website.
* [python3-htmlificate](http://repo.somethingfunny.xyz/htmlificate)
* [python3-thpath](http://repo.somethingfunny.xyz/thpath)

For more information see [the main homepage](http://repo.somethingfunny.xyz/bshs).
